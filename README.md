# Article M9 Application

This is a M9 app used in the "Node.js & Express & Pug"

## Technologies

- Node.js
- Express
- Connect Flash & Validation
- MongoDB & Mongoose
- Nodemon
- Pug Templating
- Passport.js Authentication
- Multer
- BCrypt Hashing
- Cors

### Version

1.0.1

## Usage

### Installation

Install the dependencies

```sh
$ npm install
```

Run app

```sh
$ npm start | npm run dev
```
